# AI Customer Support Chatbot (Web-Based)

A web-based AI chatbot built using Flask and SQLite. 
The chatbot handles customer queries using rule-based NLP logic 
and stores conversations in a database.

## Features
- Web-based chatbot interface
- 100+ keyword-based response conditions
- Chat history storage using SQLite
- Simple and clean frontend design
- REST API endpoint for chatbot communication

## Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000

## Project Purpose

This project demonstrates backend development, 
basic AI logic implementation, and full-stack web integration.
